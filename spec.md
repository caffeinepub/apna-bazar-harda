# Specification

## Summary
**Goal:** Replace the entire green color theme with a royal blue palette across the Apna Bazar Harda website.

**Planned changes:**
- Update CSS custom properties in `index.css` to replace all green values with royal blue (#1a4fa0), lighter blue (#2563eb) for hover/accent states, and deep navy (#0f2d6b) for dark backgrounds
- Update Tailwind config brand color tokens to the new blue palette
- Update Navigation bar background and logo accent to royal blue
- Update Hero section background/gradient overlay to royal blue
- Update all CTA and action buttons to royal blue with blue hover states
- Update Why Choose Us section icon accents and card hover highlights to blue variants
- Update Testimonials section dark background from dark green to deep navy blue
- Update Footer background from dark green to deep navy blue
- Update all section heading underlines, accent decorators, border accents, and hover states to blue variants across all component files (Navigation.tsx, HeroSection.tsx, AboutSection.tsx, WhyChooseUsSection.tsx, TestimonialsSection.tsx, GallerySection.tsx, ContactSection.tsx, Footer.tsx, App.tsx)

**User-visible outcome:** The entire website displays a consistent royal blue color theme instead of green, with white text remaining legible on blue backgrounds and white preserved for main content section backgrounds.
